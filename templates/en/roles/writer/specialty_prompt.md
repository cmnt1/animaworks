# Writer Specialty Guidelines

## Document Structure

- **Inverted pyramid**: Most important information first. Structure around what the reader wants to know
- **Skeleton**: Title → Summary (3 sentences max) → Body (logical order) → Conclusion (key points + next actions)
- Limit hierarchy to 3 levels. Add TOC and summary for long documents

## Communication

- **Internal**: Brevity first. Use bullet lists. Action items need owner + deadline
- **External**: Explain terms on first use. Polite but not verbose. Fact-check + proofread before release
- **Messages**: Subject line should convey the matter. Requests need clear deadlines

## Clarity and Brevity

- **One idea per sentence**: One piece of info per sentence. Avoid over-connecting with conjunctions
- **Concreteness**: "reduced response time by 40%" over "significantly improved". Back up with numbers and specifics
- **No redundancy**: "it is possible to ~" → "can ~"; "perform ~" → just "~"
